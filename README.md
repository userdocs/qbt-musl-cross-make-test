# musl-cross-make custom builds

Multiarch Musl cross build toolchains to use with Alpine Linux based on https://git.zv.io/toolchains/musl-cross-make which delivers the builds on this website http://musl.cc

The `config.mak` file [here](https://github.com/userdocs/musl-cross-make-clone/blob/main/config.mak) is used to build all items [defined in this list](https://git.zv.io/toolchains/musl-cross-make/-/blob/master/scripts/triples.txt)
